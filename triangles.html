To create triangles in CSS, the trick is to use a border around content that is an empty string or has no width and no height.

Then the borders on either side of the border to become a triangle must be made transparent in color.

That border with non-transparent color will angle to the virtual center of content and appear as a triangle. Nice trick!


HTML 

<div class="arrow-up"></div>

<div class="arrow-down"></div>

<div class="arrow-left"></div>

<div class="arrow-right"></div>


CSS 

.arrow-up {

    width: 0;

    height: 0;

    border-left: 10px solid transparent;

    border-right: 10px solid transparent;

    border-bottom: 10px solid blue;

}
