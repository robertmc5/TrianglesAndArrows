CSS Arrows

The triangles created with CSS border tricks can be placed at the end of other objects to create arrows or pointer-shaped structures.

One of the most common uses is shaping the navigation tree of a website. Also known as breadcrumb navigation.

An example of this use is shown above.

The code used to create these navigation arrows is shown below.


HTML 

<div class="breadcrumbs">

    <ul>

        <li><a href="index.html">Home</a></li>

        <li><a href="triangles.html">Triangles</a></li>

        <li><a href="#">Arrows</a></li>

    </ul>

    <br>

</div>


CSS 

.breadcrumbs {

    margin: 0 24px;

}

.breadcrumbs li {

    float: left;
/* avoids any extra spaces compared to display: inline-block */ 

    list-style: none;

}

.breadcrumbs a {

    color: antiquewhite;

    background: saddlebrown;

    font-weight: bold;

    text-decoration: none;

    position: relative;
/* enables offsets into the margin to take place */
    display: block;
/* combined with height, enables setting the height correctly */
    height: 24px;
/* combined with block display, sets height of breadcrumbs */ 

    line-height: 24px;

    margin-right: 24px;
/* creates the space for the arrows head and tail */ 

    padding: 0 8px;

}

.breadcrumbs a::before,
/* pseudo elements that create the arrows head and tail */ 

.breadcrumbs a::after {

    content: "";
/* the borders surround no content which enables the arrows to come to a point */
    position: absolute;
/* so the flow doesn't make space and it can fit in the margins */ 

    border-color: saddlebrown;

    border-style: solid;

    border-width: 12px 8px;
/* creates the triangle-shape for arrows */ 

}

.breadcrumbs a::before {

    left: -16px;
/* moves the tail in place in the margin */
    border-left-color: transparent;
/* creates the tail shape */ 

}

.breadcrumbs a::after{

    left: 100%;
/* enables head to be placed in the margin */ 

    border-color: transparent;

    border-left-color: saddlebrown;
/* creates the head shape */ 

}

.breadcrumbs a:hover{

    background-color: orangered;

    text-shadow: 1px 1px 1px saddlebrown;

}

.breadcrumbs a:hover::after {

    border-left-color: orangered;

}

.breadcrumbs a:hover::before {

    border-color: orangered;

    border-left-color: transparent;

}
